# Foto com Moldura - Kátia Menezes

Projeto pronto para GitHub Pages.

## Arquivos principais

- `index.html`
- `style.css`
- `script.js`
- `assets/moldura-katia-v3.png`

## Como atualizar no GitHub

1. Extraia o ZIP.
2. Entre no repositório do GitHub Pages.
3. Clique em **Add file > Upload files**.
4. Envie todos os arquivos do projeto.
5. Confirme em **Commit changes**.
6. Aguarde o deploy em **Actions**.
7. Teste em aba anônima.

## Observação

A câmera só funciona em HTTPS. O GitHub Pages já usa HTTPS automaticamente.
