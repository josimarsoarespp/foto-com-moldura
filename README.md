# Foto com Moldura para Evento

Aplicação web simples para convidados abrirem por QR Code, acessarem a câmera do celular, tirarem uma foto e baixarem a imagem com moldura personalizada.

## Arquivos

- `index.html` — estrutura da página
- `style.css` — visual da tela e moldura ao vivo
- `script.js` — câmera, captura, moldura final e download

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos para a raiz do repositório.
3. Vá em Settings > Pages.
4. Em Build and deployment, escolha Deploy from a branch.
5. Em Branch, selecione `main` e pasta `/root`.
6. Clique em Save.
7. Aguarde o GitHub gerar o link.

## Personalização rápida

No arquivo `script.js`, altere estes textos:

- `KYARA • 15 ANOS`
- `Uma noite inesquecível`

No arquivo `index.html`, altere os mesmos textos para mudar a prévia ao vivo.

## Observação importante

A câmera do celular só funciona com HTTPS. O GitHub Pages já fornece HTTPS automaticamente.
