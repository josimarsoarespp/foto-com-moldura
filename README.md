# Foto com Moldura - Kátia Menezes

Aplicação web para convidados abrirem por QR Code, acessarem a câmera do celular, tirarem uma foto vertical 9:16 e baixarem/compartilharem a imagem com moldura de girassóis.

## Arquivos

- `index.html` — estrutura da página
- `style.css` — visual da tela
- `script.js` — câmera, captura, moldura final, troca de câmera, download e compartilhamento
- `assets/moldura.png` — moldura de girassóis

## Como atualizar no GitHub

1. Extraia o ZIP no computador.
2. Abra seu repositório no GitHub.
3. Clique em `Add file > Upload files`.
4. Envie todos os arquivos do projeto, mantendo a pasta `assets`.
5. Clique em `Commit changes`.
6. Aguarde o GitHub Pages atualizar.
